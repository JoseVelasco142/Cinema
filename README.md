# Cinema
Socket.IO Interface Cinema </br>

This project is developed to understand how Node.js, express and socket.io. </br>

Components: </br>
-node.js </br>
-socket.io ^1.3.7 </br>
-express ~4.13.1 </br>

How to test: <br>
$ cd (uncompress dir_name) <br>
$ npm install <br>
$ npm start <br>
</br>
And point your browser to http://localhost:3000.
